CUM v2.0 released.

New non-shell mode.

$ CUM n Bob
The serial number of this user is 0
$ CUM v Bob 0
[OK] The user verified successfully.

How to use CUM?

  • New user
    • n/new command : n <NAME> / new <NAME>
    n Alice
    new Bob
    Output: The serial number of this user is XXX”
  • Verify user
    • v/verify command : v <NAME> <NUM> / verify <NAME> <NUM>
    v Alice 0
    verify Bob 1
    Output(SUCCESS): [OK] The user verified successfully.
    Output(FAILED): [!!] The user does not exist.”
  • Exit
    • q/quit/exit command
